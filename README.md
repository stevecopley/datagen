# DataGen

Generate fake data sets, downloadable as CSV files. Has a NZ bias as that's where I am, and it's for my role as a secondary teacher.

## Tables of Data
- Forenames (M/F)
- Surnames
- Addresses
- Creatures
- Pokemon
- Weekdays
- Months
- etc.

## Generators
- Int
- Binary int
- Hex int
- Username
- Password
- Sentences
- Words
- Date
- Time
- IPv4
- etc.

## Modifiers
- To uppercase
- To lowercase
- Left N chars
- Right N chars
- Multiply by N
- Append DATA
- Prepend DATA
